# @blackjack-deck

A comprehensive TypeScript library for building blackjack games and applications.

You can find the official documentation here: **[blackjack-deck](https://blackjack-deck.com/docs)**

## Packages

- **[@blackjack-deck/core](./packages/core)** - Core blackjack game logic and utilities
- **[@blackjack-deck/icons](./packages/icons)** - Playing card suit icons for React
- **[@blackjack-deck/react](./packages/react)** - Headless React components (coming soon)

## Installation

```bash
npm install @blackjack-deck/core components
npm install @blackjack-deck/icons
npm install @blackjack-deck/core @blackjack-deck/icons
```

## Features

- ✅ **Type-safe** - Full TypeScript support
- ✅ **Tree-shakeable** - Import only what you need
- ✅ **Zero dependencies** - Core package has no runtime dependencies
- ✅ **Headless** - Bring your own styling (React components)
- ✅ **Flexible** - Use with any framework or vanilla JS

## Development

```bash
bun install // Install dependencies
bun run build // Turborepo will build all packages
bun test // Runs each test suite
bun run dev // Starts the development server for each app and package
```
